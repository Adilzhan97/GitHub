git clone https://github.com/Adilzhan97/Java-Script.git(ссылка на репазиторий)-
создает ЛОКАЛЬНЫЙ РЕПАЗИТОРИЙ с внешнего рtпазитория на GitHub
(нужно на компьютере указать папку куда будет создаваться локальный репазиторий)
git status-проверяет статус
git commit -m "file name"
git commit -am "file name"-закомментить и добавить 
git log-вытаскивает последние логи в GitHub
git add- добавляет файлы в локальный репазиторий
git push-добавляет с локального на внешний репазиторий
git fetch-сверяет локальный и внешний репазитории на наличие всех файлов
git pull-добавляет все изменения с внешнего репазитория на локальный
(к примеру если разработчики,др тестировщики чтото комментили или добавляли в твоем репазитории)
